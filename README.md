# 🚀 Customer Relationship Management (CRM) System

## 📌 Overview

The Customer Relationship Management (CRM) System is a web-based application developed using **Python and Flask**.
It is designed to manage customer information, track leads, and streamline basic business operations.

This project was developed during a **Python Developer Internship at The Skybrisk**, focusing on practical implementation of web development concepts.

---

## ✨ Features

* 🔐 User authentication (Login system)
* 📊 Interactive dashboard for data visualization
* ➕ Add new customer records
* ✏️ Update existing customer details
* ❌ Delete records when required
* 📁 Store and manage data using SQLite
* 🖥️ Simple and user-friendly interface

---

## 🛠️ Tech Stack

* **Backend:** Python, Flask
* **Frontend:** HTML, CSS
* **Database:** SQLite
* **Development Tool:** VS Code

---

## ⚙️ System Workflow

1. User logs into the system using credentials
2. Authentication is verified by the backend
3. Dashboard is displayed with customer data
4. User can perform CRUD operations (Create, Read, Update, Delete)
5. Data is stored and retrieved from SQLite database

## ▶️ How to Run

### Prerequisites

* Python 3.x installed

### Steps

```bash
pip install flask
python app.py
```

Open browser and visit:

```
http://127.0.0.1:5000/
```
